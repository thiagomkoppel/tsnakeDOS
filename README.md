# 🐍 TSnake DOS (Python Terminal Game)

A retro-inspired Snake game built entirely in the **terminal (DOS-style)** using Python.  
This project recreates the classic Snake experience with a minimalistic, text-based interface while focusing on core game logic and efficient state management.

---

## 🎮 Features

- Classic Snake gameplay in a terminal environment  
- Grid-based movement system  
- Real-time keyboard input handling  
- Self-collision detection (game over condition)  
- Apple spawning and snake growth mechanics  
- Lightweight and minimal interface (no graphics required)  
- Retro DOS-style gameplay experience  

---

## 🧠 Technical Highlights

- Implemented using pure Python (no graphical libraries)  
- Grid-based coordinate system for movement and collision  
- Efficient state updates for snake position and growth  
- Input handling adapted for terminal-based interaction  
- Clean and modular structure for logic separation
